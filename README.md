# 老鷹抓小雞遊戲

一個使用 Pygame 開發的 2D 貪食蛇風格遊戲。

## 遊戲說明

### 目標
- 操控紅色母雞收集白色雞蛋
- 每吃一顆蛋會增加一隻黃色小雞跟隨
- 避免老鷹抓走所有小雞

### 遊戲機制

1. **母雞** (紅色方塊)
   - 使用方向鍵控制移動
   - 可以穿越邊界（從一邊出現在另一邊）
   - 不會撞牆死亡

2. **小雞** (黃色方塊)
   - 自動跟隨母雞和前方小雞
   - 形成類似貪食蛇的鏈條
   - 可以被老鷹抓走

3. **雞蛋** (白色圓形)
   - 隨機生成在地圖上
   - 吃掉後增加分數並添加一隻小雞
   - 立即生成新的雞蛋

4. **老鷹**
   - **普通老鷹** (深紅色): 追逐最後一隻小雞
   - **特殊老鷹** (紫色): 追逐最近的小雞，速度較快
   - 每 3 秒生成一隻老鷹
   - 20% 機率生成特殊老鷹

5. **難度系統**
   - 每獲得 5 分，老鷹速度提升 10%
   - 難度顯示在左上角

### 遊戲結束
- 當所有小雞都被抓走且老鷹接近母雞時遊戲結束
- 按 R 鍵重新開始

## 安裝與運行

### 安裝 Pygame
```bash
pip install pygame
```

### 運行遊戲
```bash
python chicken_game.py
```

## 操作方式

- **↑ 方向鍵**: 向上移動
- **↓ 方向鍵**: 向下移動
- **← 方向鍵**: 向左移動
- **→ 方向鍵**: 向右移動
- **R 鍵**: 遊戲結束後重新開始

## 技術特點

- 使用 `collections.deque` 實現位置歷史記錄，讓小雞平滑跟隨
- 簡單的向量計算實現 AI 追逐邏輯
- 邊界環繞機制
- 漸進式難度提升
- 雙類型敵人系統

## 遊戲畫面

- 視窗大小: 800x600 像素
- 幀率: 60 FPS
- 網格大小: 20x20 像素
